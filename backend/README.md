# Backend

Minimal Node.js + Express backend starter.

## Setup

```bash
npm install
```

## Run

```bash
npm run dev
```

or

```bash
npm start
```

The server listens on port `5000` by default. You can override it with `PORT`.

## Environment Variables

Copy `.env.example` to `.env` and update values:

- `PORT`
- `DATABASE_URL`
